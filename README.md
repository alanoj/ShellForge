## ⚡️ What is ShellForge?

**ShellForge** is a streamlined automation script designed to replicate your entire terminal setup—Zsh, Oh My Posh, custom aliases, and dev tools—onto any machine with a single command. Stop wasting hours setting up your environment; just forge it.

- 🚀 **Zero to Productive:** Go from a fresh install to your dream shell in seconds.
- 🎨 **Beautifully Themed:** Pre-configured with a custom Oh My Posh prompt.
- 📦 **Toolchain Ready:** Installs essential CLI tools (Eza, Bat, Autojump, etc.) automatically.
- 🔧 **Modular:** Easy to tweak and adapt for your specific workflow.
