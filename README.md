# 🍊 OrangeForge

OrangeForge is a curated terminal environment that brings together powerful shell enhancements, beautiful prompt themes, Git status insights, and productivity tools to create a modern, functional, and stylish CLI experience.

This repository contains all of my terminal configurations — including prompt themes, shell plugins, autocomplete tools, fonts, and helper scripts — designed for portability and ease of setup across machines.

---

## 🚀 Features

### 📍 Prompt Theme
- Custom Oh My Posh theme (`orangeforge.omp.json`)
- Colored Git information (branch, ahead/behind, stash, working/staged)
- GitHub remote icon integration when applicable
- Distinct detached branch indicator
- Execution time and last command status icons

### 🛠 Shell Enhancements
- zsh configurations with aliases and sensible defaults
- Autocomplete plugins
- Syntax highlighting
- History search

### 🔧 Developer Productivity Tools
- fzf fuzzy search
- zsh-autosuggestions
- zsh-syntax-highlighting
- Custom helper scripts and utilities
